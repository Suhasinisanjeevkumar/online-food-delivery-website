Online Food Delivery Website

A full-stack Flask-based Web Application that allows users to browse food items, manage cart, and place orders online.

🚀 Features
User registration and login system
Browse food items and restaurants
Add items to cart
Place orders
View order history
SQLite database integration

🛠️ Tech Stack
Python (Flask)
HTML
CSS
JavaScript
SQLite
Flask-Login

▶️ How to Run
1. Clone the repository
git clone https://github.com/Suhasinisanjeevkumar/online-food-delivery-website.git
2. Go to backend folder
cd online-food-delivery-website/backend
3. Install dependencies
pip install -r requirements.txt
4. Run the application
python app.py

🌐 Open in Browser
http://127.0.0.1:5000/
